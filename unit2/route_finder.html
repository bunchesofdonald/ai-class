Moving Start/Goal

To move the start or the goal, first click on the current start/goal, then click on the city you want to be the new start/goal.

Cost Functions

Each cost function will make the finder work in a slighty different way.

  • A*: Uses a straight-line heuristic combined with uniform-cost.
  • Breadth-First: Finds the shortest path, not in km, but in hops.
  • Uniform-Cost: Finds the shortest path in km.
